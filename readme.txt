Send webhooks when subscription status changes to active.

== Description ==
This plugin monitors subscription status changes and sends webhook notifications when a subscription becomes active.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/subscription-status-webhook` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Subscription Webhook screen to configure the plugin
